# PsiLabs

> Active Development

**Train intuition. Test claims. Track results. Explore consciousness with data.**

PsiLabs is an open-source platform for designing, running, and analyzing structured experiments related to anomalous perception, intuition training, blind testing, and consciousness-adjacent phenomena.

It combines the accessibility of a consumer product with the rigor of a research-minded toolkit.

# Why PsiLabs Exists

Many people are curious about topics like:

- intuition
- remote viewing
- blind perception
- anomalous guessing
- mind-matter interaction
- unusual subjective experiences

But most available tools are:

- unscientific
- poorly designed
- anecdotal only
- statistically weak
- visually outdated
- difficult to compare over time

PsiLabs aims to build the platform people wish existed:

**serious tools for unusual questions.**

# Current Focus

## Mindsight / Blind Guessing Protocols

Structured sessions involving hidden targets, repeated trials, randomized options, and measurable outcomes.

Examples:

- color target guessing
- symbol guessing
- multi-choice hidden targets
- repeat-until-correct modes
- baseline comparison sessions
